## 1. About Project
This Project is useful fo new home buyers. You can enter city name in the textbox and I will display list of available houses on sell. You can select house and look area information and more detailed information for the house.

## 2. Used APIs
1. https://docs.repliers.io/docs/repliers-developer-api/ZG9jOjMwOTc2MjE5-introduction
2. https://maps.googleapis.com/maps/api/staticmap

## 3. Some limitations using repliers real estate api        
1. https://docs.repliers.io/docs/repliers-developer-api/ZG9jOjIyMjM4OTM2-getting-started
2. For cities such as north york, missisauga, edmonton does not have any data in the api
3. For city Richmond Hill need to write city name Richmond Hill if you write richmondhill without space between it won't show any data.
